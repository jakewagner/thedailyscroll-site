# The Daily Scroll — marketing site

Public one-page site for [The Daily Scroll](https://github.com/jakewagner/thedailyscroll) (private app repo).

## Local preview

Open `index.html` in a browser, or from this directory:

```bash
python3 -m http.server 8080
```

## App Store / Play Store link

Set `STORE_URL` in `main.js` when the listing is live.

## GitHub Pages

This repo deploys from the `main` branch root (`/`).
